# bingwallpaper
必应壁纸每日自动更新下载

<hr>

**使用方法：**

双击后程序会自动更新当天的bing壁纸到桌面，然后程序自动退出。 如果要每天更新壁纸可以设置计划任务。 开始 - 管理工具 - 计划任务程序 - 点击菜单“操作” - 创建基本任务
